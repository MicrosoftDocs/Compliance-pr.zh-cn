## <a name="how-does-microsoft-365-handle-remote-access-to-production-systems"></a><span data-ttu-id="6913f-117">Microsoft 365 如何处理对生产系统的远程访问？</span><span class="sxs-lookup"><span data-stu-id="6913f-117">How does Microsoft 365 handle remote access to production systems?</span></span>

<span data-ttu-id="6913f-118">Microsoft 365 系统组件驻留在与运营团队分开的数据中心。</span><span class="sxs-lookup"><span data-stu-id="6913f-118">Microsoft 365 system components are housed in datacenters geographically separated from the operations teams.</span></span> <span data-ttu-id="6913f-119">数据中心人员不具有对 Microsoft 365 系统的逻辑访问权限。</span><span class="sxs-lookup"><span data-stu-id="6913f-119">Datacenter personnel do not have logical access to Microsoft 365 systems.</span></span> <span data-ttu-id="6913f-120">因此，Microsoft 365 服务团队人员通过远程访问管理环境。</span><span class="sxs-lookup"><span data-stu-id="6913f-120">As a result, Microsoft 365 service team personnel manage the environment through remote access.</span></span> <span data-ttu-id="6913f-121">需要远程访问以支持 Microsoft 365 的服务团队人员仅在批准了授权管理器后才向其授予远程访问权限。</span><span class="sxs-lookup"><span data-stu-id="6913f-121">Service team personnel who require remote access to support Microsoft 365 are only granted remote access after approval from an authorized manager.</span></span> <span data-ttu-id="6913f-122">所有远程访问使用 FIPS 140-2 兼容的 TLS 进行安全的远程连接。</span><span class="sxs-lookup"><span data-stu-id="6913f-122">All remote access uses FIPS 140-2 compatible TLS for secure remote connections.</span></span>

<span data-ttu-id="6913f-123">Microsoft 365 使用安全管理员工作站 (SAWs) 进行服务团队远程访问，以帮助防止 Microsoft 365 环境受到危害。</span><span class="sxs-lookup"><span data-stu-id="6913f-123">Microsoft 365 uses Secure Admin Workstations (SAWs) for service team remote access to help protect Microsoft 365 environments from compromise.</span></span> <span data-ttu-id="6913f-124">这些工作站专为防止无意或意外丢失生产数据而设计，包括锁定 USB 端口，并将所见即用的软件限制为支持环境所需的内容。</span><span class="sxs-lookup"><span data-stu-id="6913f-124">These workstations are specifically designed to prevent intentional or unintentional loss of production data, including locking down USB ports and limiting the software available on the SAW to what is required for supporting the environment.</span></span> <span data-ttu-id="6913f-125">严密跟踪和监控 SAWs，以检测并阻止 Microsoft 工程师对客户数据的恶意或意外危害。</span><span class="sxs-lookup"><span data-stu-id="6913f-125">SAWs are closely tracked and monitored to detect and prevent malicious or inadvertent compromise of customer data by Microsoft engineers.</span></span>
