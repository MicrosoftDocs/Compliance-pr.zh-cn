## <a name="integrity-checks"></a><span data-ttu-id="e1f18-172">完整性检查</span><span class="sxs-lookup"><span data-stu-id="e1f18-172">Integrity Checks</span></span>

- <span data-ttu-id="e1f18-174">**存储在元数据中的** 文件哈希：将整个文件的哈希与文件元数据一起存储，以确保在所有操作过程中保持文档级数据完整性</span><span class="sxs-lookup"><span data-stu-id="e1f18-174">**File hash stored in metadata**: Hash of the entire file is stored with file metadata to ensure document level data integrity is maintained during all operations</span></span>
